じゃんけんゲーム with チートモード

「じゃんけんゲーム with チートモード」は、JavaScriptの基礎を学んだ初学者が、 より実践的なロジック構築と状態管理を楽しく学べるWebアプリ教材です。

プレイヤーはグー・チョキ・パーの中から手を選び、 画面上の相手とじゃんけん勝負をします。

ただし、このゲームには秘密の「チートモード」が搭載されており、 モードをONにすると、自分が必ず勝てるように相手の手が操作されるという仕掛けが！

ボタンやチェックボックスでモードを切り替え、 画面に表示される勝敗や背景アニメーションなどの演出を通じて、 JavaScriptによるロジックの切り替え・状態管理・DOM操作のスキルを実践的に身につけられます
